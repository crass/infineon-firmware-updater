This function opens the logging file.

This function opens the logging file, by append an existing or create a new one. It also checks the file size, if the property storage flag PROPERTY_LOGGING_CHECK_SIZE is set. In case of the maximum file size is reached the function reopens the file by using override flag.

PppFileHandlePointer to store the file handle to. Must be closed by caller in any case.

This writes a message to the log file.

This function handles the logging work flow and writes a given message line by line to the logging file. It also checks the file size, if the property storage flag PROPERTY_LOGGING_CHECK_SIZE is set. In case of the maximum file size is reached the function reopens the file by using override flag.

Parameters
PszCurrentModuleCharacter string containing the current module name
PszCurrentFunctionCharacter string containing the current function name
PunLoggingLevelActual configured logging level
PwszMessageWide character string containing the message to log
PunMessageSizeMessage size including the zero termination
Return values

RC_E_FAILAn unexpected error occurred.
