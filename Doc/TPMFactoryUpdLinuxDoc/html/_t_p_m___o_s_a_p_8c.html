Detailed Description

Implements the TPM_OSAP command.

The module receives the input parameters, marshals these parameters to a byte array, sends the command to the TPM and unmarshals the response back to the out parameters.

This function handles the TPM_OSAP command.

The function receives the input parameters, marshals these parameters to a byte array, sends the command to the TPM and unmarshals the response back to the out parameters.

Parameters
PentityTypeThe type of entity in use
PunEntityValueThe selection value based on entityType, e.g. a keyHandle #
PsNonceOddOSAPThe nonce generated by the caller associated with the shared secret.
PpunAuthHandleHandle that TPM creates that points to the authorization state.
PpsNonceEvenNonce generated by TPM and associated with session.
PpsNonceEvenOSAPNonce generated by TPM and associated with shared secret.
