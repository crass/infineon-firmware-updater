Function Documentation

◆ FirmwareImage_Unmarshal()

_Check_return_ unsigned int FirmwareImage_Unmarshal ( _Out_ IfxFirmwareImage *  PpTarget,
_Inout_ BYTE **  PprgbBuffer,
_Inout_ INT32 *  PpnBufferSize 
)

Function to unmarshal a IfxFirmwareImage from a byte stream.

This function unmarshals the structures parameters. For all fields with variable size the output structure contains a pointer to the buffers address where the data can be found.

Parameters
PpTargetPointer to the target structure; must be allocated by the caller
PprgbBufferPointer to a byte stream containing the firmware image data; will be increased during execution by the amount of unmarshalled bytes
PpnBufferSizeSize of elements readable from the byte stream; will be decreased during execution by the amount of unmarshalled bytes
Return values
RC_SUCCESSIn case the firmware is updatable with the given firmware
RC_E_BAD_PARAMETERAn invalid parameter was passed to the function or an error occurred at unmarshal.
RC_E_BUFFER_TOO_SMALLIn case an output buffer is too small for an input byte array
