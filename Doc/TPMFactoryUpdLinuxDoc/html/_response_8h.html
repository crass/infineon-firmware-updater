◆ Response_ProgressCallback()

_Check_return_ unsigned long long Response_ProgressCallback ( _In_ unsigned long long  PullCompletion)

Callback function for progress report of EFI_FIRMWARE_MANAGEMENT_PROTOCOL.SetImage().

The function is called by EFI_FIRMWARE_MANAGEMENT_PROTOCOL.SetImage() to update the progress (1 - 100). It prints the progress to the console.

Parameters
PullCompletionProgress completion value between 1 and 100
Return values
0The callback executed successfully.
