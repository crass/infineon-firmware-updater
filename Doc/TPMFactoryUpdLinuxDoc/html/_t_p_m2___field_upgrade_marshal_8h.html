Detailed Description

Declares methods to marshal and unmarshal TPM1.2 and TPM2.0 commands, structures and types.

This module provides marshalling and unmarshalling function for all TPM1.2 and TPM2.0 commands, structures and types defined in the Infineon Secure Boot Loader Specification.
