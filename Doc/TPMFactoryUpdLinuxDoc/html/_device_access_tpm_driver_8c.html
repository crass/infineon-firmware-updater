◆ DeviceAccessTpmDriver_Initialize()

_Check_return_ unsigned int DeviceAccessTpmDriver_Initialize ( )

Initialize the device access via config setting DEVICE_PATH.

Default value is /dev/tpm0. If an invalid device path is configured the tool will return "No connection to the TPM or TPM not found (0xE0295200)".

Return values
RC_SUCCESSThe operation completed successfully.
RC_E_INTERNALThe operation failed.
RC_E_NO_TPMIn case of an open call to /dev/tpm0 failed

TPM transmit function.

This function submits the TPM command to the underlying TPM.

Parameters
PrgbRequestBufferPointer to a byte array containing the TPM command request bytes
PunRequestBufferSizeSize of command request in bytes
PrgbResponseBufferPointer to a byte array receiving the TPM command response bytes
PpunResponseBufferSizeInput size of response buffer, output size of TPM command response in bytes
Return values
RC_SUCCESSThe operation completed successfully.
RC_E_BAD_PARAMETERAn invalid parameter was passed to the function.
RC_E_INTERNALIf the DEV_TPM_HANDLE is not present in the property PropertyStorage
RC_E_FAILAn unexpected error occurred.
